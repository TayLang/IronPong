## Prerequisites

### Install mongodb (once)

  - `brew install mongodb`
  - `sudo mkdir -p /data/db`
  - `sudo chown -R $(whoami) /data/`

### Installing

```
# install node_modules (one-time)
npm install
```
```
#run local server
npm run go 
```
